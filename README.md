# Family Home Mini App

Telegram Mini App для семейного управления бытовыми задачами и списком покупок.

## Что внутри

- каталог бытовых задач по дому
- каталог продуктов для регулярных покупок
- общий список текущих задач для обоих пользователей
- общий список продуктов, которые скоро закончатся или уже закончились
- отметка задач как выполненных
- отметка продуктов как купленных
- хранение данных в Postgres через Prisma

## Локальный запуск

1. Создай `.env` по примеру `.env.example`.
2. Укажи `DATABASE_URL`.
3. Установи зависимости:

```bash
bun install
```

4. Сгенерируй Prisma client и примени схему:

```bash
bunx prisma generate
bunx prisma db push
```

5. Запусти приложение:

```bash
bun run dev
```

## Как это работает

- При первом запросе приложение автоматически добавляет базовые каталоги задач и продуктов.
- Любой пользователь, открывший Mini App, видит общий актуальный список.
- Telegram WebApp используется для получения имени текущего пользователя внутри Mini App.

## Подключение как Telegram Mini App

У бота через `@BotFather`:

1. Создай бота через `/newbot`, если его еще нет.
2. Открой `/mybots` -> выбери бота -> `Bot Settings` -> `Menu Button`.
3. Укажи URL задеплоенного приложения.

## Деплой во Vercel

```bash
vercel deploy . -y
```

В Vercel добавь переменную окружения:

- `DATABASE_URL`

После этого выполни redeploy.

## База данных

Проект рассчитан на Postgres или Neon.

Если схема еще не применена:

```bash
bunx prisma db push
```

Если позже захочешь историю миграций:

```bash
bunx prisma migrate dev --name init
```

## Что можно докрутить дальше

- отдельные семейные пространства или несколько household-профилей
- фильтры по комнатам, категориям и срочности
- push-уведомления в Telegram при новых задачах и покупках
- историю закрытых задач и купленных продуктов
